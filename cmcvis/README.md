# Coincidental Multicast Tree Visualization

This repository contains a simple R Shiny tool to visualize and experiment with the coincidental multicast (CMC) gains achievable by the POINT IP-over-ICN technology in different scenarios. The R packages needed to run the tool are shiny, shinythemes, VGAM, spatstat, igraph, and Cairo. For convenience the shell script "runDemo.sh"Äù is also provided to run the tool.
